nuestro trello si no tiene cuenta cree una https://trello.com/invite/b/698b44ae8640f7b28544b957/ATTI5a76b020fe4a9ba2d9a79ec03310b7573800DD30/veraff


Proyecto desarrollado con:

Backend principal: Node.js + Express

Servicio de autenticación: C# .NET 8 (AuthService)

Base de datos: PostgreSQL / MongoDB (según configuración)

Autenticación: JWT

Antes de ejecutar el proyecto debes tener instalado:

Node.js

Postman

.NET 8 SDK

PostgreSQL (si aplica)

Docker 

de primero al momento de ejecutar el programa para los endPoint de primero necesitas descargar dependecias para intalarlas en la terminal del Visual Studios Code o en la de tu computadora (pero debes de estar en el proyecto) debes escribir pnpm install y se descargaran luego debes de poner node index.js y aparecera esto

http://localhost:3001/veraff/v1

Con esto puedes entrar a Postman y iniciarlo los ENDPOINT

http://localhost:3001/veraff/v1/cards
